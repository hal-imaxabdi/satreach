# SatReach
### Satellite Coverage Explorer — SHP 2026 ORBIT Hackathon

A browser-based interactive map that visualises the internet connectivity
gap in remote regions and simulates what LEO satellite coverage could look like.

## How to run
Just open index.html in your browser. No installation needed.

## Tech used
- Leaflet.js (interactive maps)
- Vanilla HTML, CSS, JavaScript
- Data: ITU / World Bank 2023
